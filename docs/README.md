# JDM Club Manager

Application open source de gestion pour La Jeunesse du Marais.

## Objectifs

- Gestion des adhérents
- Gestion des groupes
- Gestion des paiements
- Gestion des présences
- Gestion des compétitions
- Boutique du club
- Notifications
- Gestion documentaire

## Planning

- Saison 2025-2026 : Conception
- Saison 2026-2027 : Développement et tests
- Saison 2027-2028 : Mise en production

## Club pilote

La Jeunesse du Marais - Lomme

- Club fondé en 1905
- Gymnastique Artistique Masculine
- Baby Gym mixte
- Gym adulte mixte
- Renforcement musculaire
- Environ 300 adhérents
