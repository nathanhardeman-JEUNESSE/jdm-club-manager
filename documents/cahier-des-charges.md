# Cahier des charges

Projet : JDM Club Manager

Application open source de gestion pour La Jeunesse du Marais.

## Objectif

Créer une plateforme permettant de gérer :

- les adhérents
- les familles
- les groupes
- les présences
- les paiements
- les compétitions
- la boutique
- les notifications

Mise en production visée : Saison 2027-2028.
