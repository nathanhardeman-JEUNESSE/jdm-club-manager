# Charte graphique - JDM Club Manager

## Objectif

Définir l'identité visuelle de l'application JDM Club Manager.

L'application doit donner une impression :

- sérieuse
- qualitative
- sobre
- moderne
- accessible

---

# Couleurs

## Couleurs principales

- Bleu JDM : `#0057FF`
- Rouge JDM : `#E30613`

## Couleurs de fond

- Anthracite : `#262626`
- Noir carte : `#161616`

## Couleurs de texte

- Blanc : `#FFFFFF`
- Gris clair : `#C8C8C8`

---

# Style général

L'application utilise principalement un thème sombre.

Le bleu JDM sert aux actions principales.

Le rouge JDM sert uniquement aux alertes, notifications importantes ou éléments à mettre en valeur.

Les écrans doivent rester sobres, lisibles et aérés.

---

# Logo

Le logo officiel de La Jeunesse du Marais doit être utilisé sur les écrans principaux.

Il doit rester lisible et ne pas être affiché trop grand.

Sur la page d'accueil publique, le logo peut être placé sur une photo de la salle.

---

# Photos

Les photos doivent être utilisées uniquement lorsqu'elles renforcent l'identité du club.

Utilisation recommandée :

- accueil public
- page Mon club
- présentation du club

Les écrans fonctionnels doivent rester simples et sombres.

---

# Cartes

Les informations importantes sont affichées sous forme de cartes.

Une carte correspond à une action ou une information principale.

Règle :

Une carte = une action.

---

# Boutons

Les boutons principaux utilisent le bleu JDM.

Au survol ou pour attirer l'attention, le rouge JDM peut être utilisé.

Les boutons doivent être simples, lisibles et faciles à toucher sur mobile.

---

# Navigation

La navigation doit rester simple.

L'utilisateur doit pouvoir comprendre où aller sans explication.

L'application ne doit pas devenir une usine à gaz.

---

# Responsive

L'application est développée selon le principe "Mobile First".

Tous les écrans sont conçus en priorité pour une utilisation sur smartphone.

Les versions tablette et ordinateur sont des adaptations de la version mobile.

Les cartes utilisent les règles suivantes :

- Coins arrondis : 18 px
- Ombre légère
- Fond noir (#161616)
- Une seule action par carte
- Espacement vertical régulier entre les cartes

# Espacements

Les marges et espacements doivent rester cohérents.

Références :

- Marge extérieure : 18 px
- Espacement entre les cartes : 20 à 25 px
- Padding interne des cartes : 22 px

# Typographie

Titre principal (H1)

- 36 px
- Gras

Titre de carte (H2)

- 28 px
- Gras

Texte courant

- 16 px

Texte secondaire

- Gris clair

# Philosophie

Chaque écran doit répondre à trois critères :

- Beau
- Simple
- Rapide à comprendre

Si une fonctionnalité complique inutilement l'application, elle doit être repensée avant d'être développée.
